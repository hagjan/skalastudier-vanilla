Programvare

  • Slåtteplukkaren

    Slåtteplukkaren er eit verktøy som kan brukast til analyse og transkribering og diverse anna skreve av Morten Joh. Tonegeneratoren inne på skalasidene er basert på Slåtteplukkaren. Verktøyet / patchen opnast i SuperCollider og både SuperCollider og Slåtteplukkaren er open source programvare som kan lastast ned gratis.

  • Transcribe!

    Eit av fleire program som er hendauge når ein vil finlytte til arkivopptak ved å dra ned tempoet, fryse tonar, justere grunntone dersom ein spelar eller syng attåt osb. Det er fleire gode program så berre vel det som funkar best for ditt føremål.
